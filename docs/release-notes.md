## v0.1.0 — MVP (2026-02-28)

「投げる・振る・当たる」の基本ループが完成。

### 遊べる内容
- 10球のストレートを打ち返し、3本以上ホームランを打てばステージクリア
- PC: 矢印キーで移動、スペースでスイング
- スマホ: 左半分ドラッグで移動、右半分タップでスイング

### 主な特徴
- パワーゲージ（心拍システム）: 鼓動のタイミングに合わせてスイングするとパワーアップ
- スイートスポット: バットの芯で捉えると飛距離が伸びる
- 3コース（内角・真ん中・外角）からランダム投球

### 既知の制限
- 球種はストレートのみ
- サウンドなし
- ステージは1つのみ
