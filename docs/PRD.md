# プロジェクト：HONEY-DERBY

## 概要
『くまのプーさんのホームランダービー！』の独特な操作感と、ファミスタ風の縦型視点を融合させた野球ゲーム。
シンプルながらも、「芯」で捉える快感と理不尽な魔球に挑む中毒性を段階的に実装する。

---

## 開発マイルストン

### マイルストン 1: MVP (Minimum Viable Product) - 打撃の核心
- **目的**: 「投げる・振る・当たる」の基本ループの実装。
- **視点**: 縦型 Canvas。奥にピッチャー、手前にバッターを配置。
- **投球シーケンス**: 
  - ピッチャー頭上に「3, 2, 1」とカウントダウンを表示。
  - カウント終了直後にボールを発射（まずはストレートのみ）。
- **バッター挙動**: 
  - `Left / Right` キーで左右移動。
  - `Space` キーでスイング（振り抜きと戻りのクールタイムを設定）。
- **判定ロジック**: 
  - **方向**: Y軸の通過タイミング（Early / Just / Late）で決定。
  - **飛距離**: インパクト時のX軸のズレ（バットの芯とボールの距離）で決定。
- **出力**: 飛距離（m）と「HOME RUN / HIT / OUT」を画面に表示。

### マイルストン 2: 戦略性の導入（バッターボックスの自由と変化球）
- **目的**: 立ち位置の調整と球種への対応。
- **仕様**:
  - **自由移動**: 左右キーでバッターボックス内を自在に移動可能にする。
  - **変化球実装**: ピッチャーが「カーブ」「シュート」「緩急」を織り交ぜるロジック。
  - **判定調整**: バッターの立ち位置に応じて、最適なインパクトタイミングが変化するように計算。

### マイルストン 3: 演出の強化と 1st ステージの完成
- **目的**: 視覚的な快感とゲームサイクルの成立。
- **仕様**:
  - **打球アニメーション**: 打ったボールが奥へ小さくなりながら飛んでいく演出。
  - **演出効果**: ホームラン時の画面の揺れ、リザルトテキストの装飾。
  - **ノルマシステム**: 規定投球数と目標本数を設定し、クリア判定（STAGE CLEAR）を実装。

### マイルストン 4: プログレッションと育成システム
- **目的**: 継続プレイの動機付け。
- **仕様**:
  - **タイトル画面**: 『HONEY-DERBY』ロゴを表示。
  - **ステージ選択**: 複数の対戦相手を選べるメニュー。
  - **ステータス強化**: 獲得ポイント（ハチミツ）を消費し、パワー、ミート、スピードを強化。

### マイルストン 5: 伝説の「魔球」ステージ
- **目的**: 理不尽な難易度の再現。
- **仕様**:
  - **魔球アルゴリズム**: 上下蛇行、急加速、消える魔球、そして全能力を持つ最終ボスの実装。
  - **最終調整**: プレイヤーがフル強化状態でも苦戦する絶妙なバランス調整。

## 技術仕様
- **言語**: JavaScript (Vanilla), HTML5 Canvas, CSS3
- **操作**: キーボード（左右矢印：移動、Space：スイング）
- **資産管理**: 画像素材は `assets/` フォルダに配置（bat.png, ball.png 等）
