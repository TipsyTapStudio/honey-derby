# ブログネタ帳

devlog を書くたびに「外部の読者に刺さりそうな話」をここにストックする。
セッション終了時に Claude Code が devlog を振り返り、ネタ候補を追記する。

## ジャンル凡例

| タグ | 想定読者 | 掲載先の例 |
|------|---------|-----------|
| `#eng` | エンジニア（JS/Canvas/ゲーム開発） | Zenn, Qiita |
| `#design` | デザイナー・UI/UX | note, Zenn |
| `#gamedesign` | 企画・ゲームデザイナー | note |
| `#ai-dev` | AI活用・開発プロセスに興味がある人 | note, Zenn, Qiita |

## ステータス凡例

| マーク | 意味 |
|--------|------|
| `- [ ]` | ネタ出し段階 |
| `- [~]` | ドラフト執筆中 → `docs/blog/{slug}.md` にリンク |
| `- [x]` | 公開済み → 公開URLを添える |
